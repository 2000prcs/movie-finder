## Usage

From within the root directory:

1. Install project dependencies:

```sh
npm install
```

2. Build a client bundle with Webpack: 

```sh
npm run build 
```

3. Start Express server:

```sh
npm start 
```

4. Open browser at `127.0.0.1:7070` 

5. Enjoy!
